      # 必应每日一图

      # 官方

      接口地址：https://cn.bing.com/HPImageArchive.aspx?format=xml&idx=0&n=1 (opens new window)

      参数说明

      参数 性质 说明
      format 可选 js | xml , 默认为 xml
      idx 可选 取值范围 [-1, 7] , 分别表示 明天，今天，昨天，前天…… ， 默认为 0
      n 可选 取值范围 [1, 8] , 表示从 idx 指定的日期 往前推 n 张图片的信息

      例如 idx 为 7， n 为 8 返回结果如下

      # 非官方

      1. 必应壁纸接口 (opens new window)

        随机必应图片的地址 https://bing.ioliu.cn/v1/rand?w=1920&h=1200 (opens new window)

             <img src="https://bing.ioliu.cn/v1/rand?w=1920&h=1200"/>
        1

      2. 必应壁纸api (opens new window)

        可以直接在 img 中使用的 地址 https://bing.creepersan.com/api/v1/img-current (opens new window) , 分辨率为 1920 × 1080

            <img src="https://bing.creepersan.com/api/v1/img-current" />
